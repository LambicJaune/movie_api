The Tom Hanks Movies API

An API that returns data about movies featuring Tom Hanks : complete list of movies involving the actor, movie description, genre (retrieved by name, e.g., "Comedy"), director (including data, e.g. bio, birth year, death year), image URL, whether the movie if featured or not. Users can register as well as deregister, update their user information, and add or remove movies from their list of favorites

Users Endpoints

Request URL Endpoint HTTP Method Query Parameters Request body data format Response body data format
Register a new user /users POST None A JSON object holding data about the user:

{
"userName": "john_doe",
"email": "john@example.com",
"password": "PW123!"
} 		A JSON object holding data about the user including a generated ID and FavoriteMovies array (as well as the document's version):

{
"userName": "john_doe",
"email": "john@example.com",
"password": "PW123!"
"_id": "4596848964896qfd464d1"
"__v": 0 }
Update user information /users/:userName PUT Current userName None A text message indicating the updated user information. Example :

{ "Username": "Jane Doe",
"Password": "securePass456",
"Email": "john@example.com",
"Birthday": "1990-01-01",
"FavoriteMovies": ["68639866fdce14bfc0748a5f"] }
Add a movie to user’s favorites /users/:userName/favorites/:movieID POST userName and movieID as URL param None A JSON object containing the user's info including the ID of the movie added to FavoriteMovies array. Exemple :

{ "_id": "68658c5982a55aa80fab737e",
"Username": "John Doe",
"Password": "securePass456",
"Email": "john@example.com",
"FavoriteMovies":
["68639866fdce14bfc0748a5f"],
"__v": 0}
Remove a movie from user’s favorites /users/:userName/favorites/:movieID DELETE userName and movieID as URL param None A JSON object containing the user's info including a FavoriteMovies array displayed without the ID of the movie that was just deleted. Example :

{ "_id": "68658c5982a55aa80fab737e",
"Username": "John Doe",
"Password": "securePass456",
"Email": "john@example.com",
"FavoriteMovies": [],
"__v": 0}
Deregister an existing user /users/:userName DELETE userName as URL param None A text message indicating: "[Username] was deleted"
