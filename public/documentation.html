The Tom Hanks Movies API

An API that returns data about movies featuring Tom Hanks : complete list of movies involving the actor, movie description, genre (retrieved by name, e.g., "Comedy"), director (including data, e.g. bio, birth year, death year), image URL, whether the movie if featured or not. Users can register as well as deregister, update their user information, and add or remove movies from their list of favorites

Movies Endpoints

Request URL Endpoint HTTP Method Query Parameters Request body data format Response body data format
Returns a list of all movies /movies GET None None An array of objects (movies)
Returns movie details by title /movies/:title GET Movie title None A movie object displaying title, genre, director, description, imageURL, if featured of not
Returns movies by genre name /genres/:genreName GET Genre name None An array of objects containing all the movies of the selected genre
Returns director information by name /directors/:directorName GET Director's name None An object containing a director's data including bio, birth year, death year

Users Endpoints

Request URL Endpoint HTTP Method Query Parameters Request body data format Response body data format
Register a new user /users POST None A JSON object holding data about the user:

{
"username": "john_doe",
"email": "john@example.com",
"password": "PW123!"
} 		Confirmation message in plain text or created user object
Update user information /users/:username PUT Current username None A text message indicating the updated user information
Add a movie to user’s favorites /users/:username/favorites/:movieID POST Username, movieID None A text message indicating: "Movie added to your favorites"
Remove a movie from user’s favorites /users/:username/favorites/:movieID DELETE Username, movieID None A text message indicating: "Movie removed from your favorites"
Deregister an existing user /users/:id DELETE Username None A text message indicating: "You have been deregistered"
